# Wedding Halls Booking

Wedding Halls Booking is a simple front-end web project created for learning how to work with Trello, GitHub, and team collaboration.

The project focuses on building four basic pages for a wedding halls booking website. It does not include a real backend or database. The main goal is to practice task management, assigning work between team members, and using GitHub for sharing code.

## Project Idea

The idea of the project is to create a simple website where users can view wedding halls, book a hall, and where an admin can simulate adding a new hall.

## Project Pages

The project contains four main pages:

1. **Login Page**
   - Allows the user to enter the website using a simple login form.

2. **Show Halls Page**
   - Displays a list of available wedding halls using static/fake data.

3. **Book A Hall Page**
   - Allows the user to fill in a booking form for a selected hall.

4. **Add A Hall Page**
   - Allows the admin to fill in a form to simulate adding a new wedding hall.

## Project Features

- Simple and clean front-end design
- Login form page
- Display wedding halls as cards
- Booking form
- Add hall form
- Static/fake data only
- Basic form validation
- Responsive page layout
- Team task management using Trello
- GitHub collaboration practice

## Technologies Used

## Trello Workflow

We used Trello to organize the project tasks.

The board contains lists such as:

- Backlog
- To Do
- Doing
- Review
- Done

Each team member is assigned to one task/card.

## Team Tasks

| Task | Description |
|---|---|
| Login Page | Design and implement the login page |
| Show Halls | Display available wedding halls |
| Book A Hall | Create the booking form page |
| Add A Hall | Create the add hall form page |

## Project Goal

The goal of this project is not to build a complete real booking system, but to learn:

- How to divide a project into tasks
- How to write user stories
- How to use Trello for project management
- How to assign tasks to team members
- How to collaborate using GitHub
- How to build simple front-end pages

## Notes

This project is for learning purposes only.  
There is no real authentication, no database, and no backend connection.
